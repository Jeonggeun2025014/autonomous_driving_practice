GitHub Actions skeleton (manual steps required)

This file describes steps for a CI workflow to build the Android project using Bubblewrap and Android SDK. It is a high-level skeleton; secrets for keystore and SDK config are needed.

1) Install Java and Android SDK on runner
2) Install Node and Bubblewrap
3) Run bubblewrap init/build
4) Use Gradle to assemble an AAB
5) Sign AAB with keystore stored in GitHub secrets

I can create a full `workflow.yml` if you want to automate CI/CD — tell me if you want that.
